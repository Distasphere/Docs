Welcome to flexy-payments documentation!
===================================

**Flexy-payments** is an innovative multichannel payment aggregation solution designed to unify all financial services in West Africa. Flexy enables individuals, merchants, businesses, and government institutions to access, from a single point, all modern payment methods: mobile money, bank cards, USSD, transfers, microfinance, and open APIs.

.. note::

   This project is under active development.
